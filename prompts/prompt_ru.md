Роль
Сгенерируй полный исполняемый Python-скрипт для обнаружения ботнет-атак в IoT-трафике методом поиска аномалий. Код должен быть хорошо прокомментирован, модульно организован и выводить понятный прогресс выполнения.

Контекст
Нужно обнаруживать ботнет-атаки с помощью глубокого автоэнкодера, обученного только на нормальном IoT-трафике. Подход основан на работе Meidan et al. (2018) по датасету N-BaIoT: ошибка реконструкции используется как оценка аномальности. Если MSE реконструкции выше заданного порога, трафик считается атакой.

Данные
Используй датасет N-BaIoT из UCI Machine Learning Repository, ID 442. Он содержит реальный сетевой трафик от 9 IoT-устройств, включая камеры, дверные звонки и термостаты. Устройства заражались двумя ботнетами: Mirai и BASHLITE/Gafgyt.

Каждая запись CSV содержит 115 числовых признаков — статистик сетевых потоков.

Файлы организованы так:

<device_id>.benign.csv — нормальный трафик
<device_id>.gafgyt.<attack>.csv — атаки BASHLITE: combo, junk, scan, tcp, udp
<device_id>.mirai.<attack>.csv — атаки Mirai: ack, scan, syn, udp, udpplain
